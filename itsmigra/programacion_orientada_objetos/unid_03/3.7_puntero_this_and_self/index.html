3.7 Puntero this y puntero self

En POO en ocasiones es necesario hacer referencia al objeto en cuestión, para esto los lenguajes de programación hacen uso de alguna referencia hacia si mismo, de esta forma que this (en Java) o self(en Python) está disponible cuando un método es invocado dentro del contexto de un objeto.

Esta referencia hace alusión objeto invocador (el objeto al cual el método pertenece).

3.7.1 Ejemplos:

Para que quede mas claro veamos un ejemplo a partir de la clase Alumno vista previamente, pero versión simplificada para fines demostrativos:

3.7.2 Referencia this en Java.

veamos como quedaría este ejemplo en Java:

public class Alumno {
	private String nombre;
	// Constructor
	public Alumno(String name) {
		this.nombre = name;
	}
	protected void setNombre(String n) {
		nombre = n;
	}
	public String getNombre() {
	  return nombre;
	}
	//Probando ejemplo
	public static void main(String args[]) {
		Alumno j = new Alumno("Juan");
		Alumno p = new Alumno("Pedro");
		System.out.println(j.getNombre());
		System.out.println(p.getNombre());
	}
}

Como podemos ver la clase Alumno tiene el atributo nombre, de esta clase generamos dos instancias(Objetos), cada uno con su propio valor “Juan” y “Pedro” al cual accedemos por medio del atributo this.nombre.

Usted puede descargar y probar el código: Alumno.java

3.7.3 Referencia self en Python.

En el lenguaje Python la referencia self tiene un comportamiento análogo a this en Java, veamos como quedaría el mismo ejemplo con python.

class Alumno:
	''' Constructor '''
	def __init__(self, name):
		self.__nombre= name
	''' Método setNombre '''
	def setNombre(self, name):
		self.__nombre= name
	''' Método getNombre '''
	def getNombre(self):
		return self.__nombre

# Probando ejemplo
j = Alumno("Juan")
p = Alumno("Pedro")
print(j.getNombre())
print(p.getNombre())
