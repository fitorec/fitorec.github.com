3.6.1 Creancion de objetos con Java:

Con java para definir una clase se ocupa la palabra reservada class una clase es un segmento de código por lo cual se encuentra delimitada por los separadores de segmento de código({}), veamos como quedaría esta clase con solamente los atributos:

public class Alumno {
	public String matricula;
	private String nombre;
}

Notar que la definción de atributos tienen la siguiente estructura

ModificadorAcceso TipoDato nombreAtributo;

De forma análoga podemos definir métodos basandonos en la siguiente estructura:

	ModificadorAcceso TipoDatoRetorno nombreMetodo(TipoArg1 arg1,...){
		//Código del método
	}

Al agregar métodos la clase quedaría como:

public class Alumno {
	public String matricula;
	private String nombre;
        // Métodos
        protected void setMatricula(String m) {
            matricula = m;
        }
        public String getMatricula() {
            return matricula;
        }
        protected void setNombre(String n) {
            nombre = n;
        }
        public String getNombre() {
            return nombre;
        }
        private void darDeBaja() {
            // Código de Baja para un Alumno
        }
}

3.6.1.2 Prueba de objetos con Java

Una vez creadas las clases es necesarias instanciarlas a una instancia de una clase se conoce como objeto, para crear una instancia de una clase en java se usa la palabra reservada new como se puede ver a continuacion:

Alumno juan = new Alumno();
juan.setNombre("Juan Garcia");
System.out.println(juan.getNombre());
// imprime:
//	Juan Garcia
