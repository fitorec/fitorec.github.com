2.0 arreglos

Un arreglo es un contenedor de un conjunto de datos, a los que podemos acceder por medio de un indice,

Arreglos en Java.

Los arreglos en java tiene una serie de características:

  • Un arreglo solo puede contener elementos de un mismo tipo.
  • Un arreglo es un Objeto, esto implica que tiene tanto métodos como atributos.
  • El tamaño de un arreglo es fijo y se determina en el momento que se inicializa.
  • Un arreglo de tamaño n sus indices validos son del 1 al n-1.

Ejemplo declarando un arreglo:

tipoDato[] nombreArreglo;   // Sintaxis recomendada.
tipoDato nombreArreglo[];  // Funciona pero no se recomienda.

Como podemos ver es valido declarar un arreglo sin tamaño, posteriormente definir el tamaño al momento de inicializar la variable nombreArreglo, como se muestra a continuación.

nombreArreglo = new tipoDato[Tamaño];

Veamos como quedaría un ejemplo para un tipo de dato especifico(int) y de un tamaño especifico 5:

int[] arregloNumerico;
arregloNumerico = new int[5];

De forma análoga podríamos definir arreglos de cualquier tipo:

byte[] unArregloDeBytes;
short[] unArregloDeShorts;
long[] unArregloDeLongs;
float[] unArregloDeFloats;
double[] unArregloDeDoubles;
boolean[] unArregloDeBooleans;
char[] unArregloDeChars;
String[] unArregloDeStrings;

Asignación de datos:

Una vez creado y definido el arreglo podemos asignarle valores a sus elementos:

int[] arregloNumerico;
arregloNumerico = new int[5];
// Asignación
arregloNumerico[0] = 8;
arregloNumerico[1] = 7;
arregloNumerico[2] = 6;
arregloNumerico[3] = 5;
arregloNumerico[4] = 4;

Ahora podemos acceder a cada elemento por su posición:

System.out.println(arregloNumerico[0]) // imprime 8;
System.out.println(arregloNumerico[1]) // imprime 7;
System.out.println(arregloNumerico[2]) // imprime 6;
System.out.println(arregloNumerico[3]) // imprime 5;
System.out.println(arregloNumerico[4]) // imprime 4;

Una forma alternativa de inialización y asignación es la sintaxis reducida, como se muestra acontinuación:

int arregloNumerico[] = {8,7,6,5,4};
System.out.println(arregloNumerico[0]) // imprime 8;

Arreglos en python.

miArreglo=[1,2,3,4,5,6]
print miArreglo[2]
miArreglo[2]=100

Podemos extraer una parte del arreglo:

print miArreglo[2:5]
# [3,4,5]
print miArreglo[2:]
# [3,4,5,6]
print miArreglo[:3]
# [1,2,3]
print miArreglo[:]
# [1,2,3,4,5,6]
